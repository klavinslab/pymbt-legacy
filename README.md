# pymbt

Tools for molecular (and synthetic) biologists

### Requirements:

python (pip-compatible):
------------------------
> numpy  
> biopython  
> matplotlib  

system:
-------
for matplotlib

> libpng dev headers
> libfreetype dev headers
> backend headers (e.g. tk-dev, tcl-dev)

for needleman-wunsch alignment
> emboss

### Clone pymbt into your current directory, then:
> cd pymbt  
> pip install .
