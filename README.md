# pymbt

PyMBT: Python molecular biology tools.

## Requirements:

###python (pip-compatible):

```
numpy  
biopython  
matplotlib  
```

###system:

for matplotlib

```
libpng-dev
libfreetype-dev 
tk-dev
tcl-dev
```

for needleman-wunsch alignment

```
emboss
```

## Clone and install:

```
git clone https://github.com/klavinslab/pymbt.git
cd pymbt  
pip install .
```
